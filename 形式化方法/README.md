### 复习建议

**考前把 PPT 上的知识点过一遍就可以** —— 复习以课件为主，建议：

1. **通读 PPT**：至少完整过一遍所有课件，建立知识框架
2. **重点标记**：标记老师强调的重点和高频考点
3. **理解概念**：搞清楚基本定义、定理和推理规则
4. **例题练习**：弄懂课件上的例题和推导过程
5. **查漏补缺**：针对不懂的地方问AI
