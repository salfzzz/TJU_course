# 数据库系统


---

## 📚 课程简介

数据库系统是计算机专业的核心课程，主要讲授数据库的基本原理、设计方法和应用技术，包括关系数据库、SQL 语言、数据库设计、事务管理等内容。

## 🧪 实验相关

### 实验数量
**实验比较多**，本课程包含多个实验项目，覆盖数据库设计、SQL 编程、数据库管理等多个方面。

### 实验特点
**主要是跟着实验指导一步步复现** —— 实验都有详细的指导文档，按照步骤操作即可完成。建议：

- **认真阅读实验指导**：实验前先通读指导文档，了解实验目的和步骤
- **逐步操作复现**：按照指导一步步执行
- **完成后回顾总结**：实验结束后梳理知识点，考试中都是类似的指令


## 📝 考试相关

### 复习重点

#### 1. 关注 PPT 知识点
**考试主要要关注 PPT 上的知识点** —— 考试内容大多来自课堂 PPT，建议：

- PPT 至少过 **2-3 遍**
- 重点关注老师强调的内容
- 整理 PPT 中的概念、过程
- 标记不懂的地方，多问AI比什么都好用

#### 2. 多看往年题
**多看几遍往年题** —— 往年题是最好的复习资料：

- 熟悉考试题型和难度
- 找出常考知识点和高频考点
- 总结答题思路和技巧
- 模拟考试，控制答题时间

### 常见题型
- 选择题 / 填空题 —— 基础概念
- 简答题 —— 原理和方法
- SQL 编程题 —— 查询语句编写
- 设计题 —— ER 图、关系模式设计
- 综合题 —— 事务、并发等

## 💡 学习建议


### 复习方法
1. **第一轮**：先看 PPT，建立整体框架
2. **第二轮**：做往年题，查漏补缺
3. **考前**：快速过一遍重点和错题
