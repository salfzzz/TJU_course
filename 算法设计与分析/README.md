# 算法设计与分析

---

## 📚 课程简介

算法设计与分析，主要讲授经典算法的设计思想和分析方法，包括贪心、分治、动态规划、回溯、分支限界等多种算法，以及时间复杂度、空间复杂度的分析方法。

## 📝 考试相关

### 难度评估

**本课程考试难度很大** —— 考试题目灵活多变，不是靠背就能过的课。

### 题目特点

**会涉及很多 PPT 上没有的题目** —— 考试不会直接考 PPT 原题，而是会出各种变形题和应用题：


> 💡 **不过总体算法都是里面的** —— 虽然题目不是原题，但用到的算法思想和方法基本都在课件范围内，核心算法都在 PPT 里。

### 常见考点（参考）
- 时间复杂度分析（最好、最坏、平均）
- 贪心算法（活动选择、背包问题等）
- 分治算法（归并排序、快速排序等）
- 动态规划（最长公共子序列、背包问题等）
- 回溯法（子集和等）
- 分支限界法
