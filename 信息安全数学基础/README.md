# 信息安全数学基础

## 📚 课程简介

信息安全数学基础是信息安全专业的核心基础课程，主要讲授密码学和信息安全领域所需的数学理论与方法。

## ⚖️ 难度评估

### 整体难度
**课程难度不低**，

### 25-26 学年难度变化
**相对简单不少** —— 2025-2026 学年的课程内容**未涉及代数系统部分知识点**，相比往年减少了较为抽象难懂的代数结构内容，学习压力有所降低。

> 💡 **提示**：虽然难度有所下降，但核心的数论部分仍然是重点和难点，不可掉以轻心。

## 📖 主要知识点

### 数论基础
- 整除与最大公约数
- 同余理论
- 欧拉定理与费马小定理
- 模逆元与扩展欧几里得算法
- 中国剩余定理
- 二次剩余与勒让德符号
- 原根与离散对数

### 其他内容
- 有限域基础
- 概率论基础
- 信息论初步

> ✅ **说明**：25-26 学年未涵盖代数系统（群、环、域等抽象代数结构）的深入讲解。

## 💡 学习建议

### 1. 多练习是关键
**需要多练习掌握** —— 数学课程光看不练很难真正掌握。建议：
- 每学完一个知识点，及时完成课后习题
- 重点练习计算类题目（如求逆元、解同余方程等）
- 整理典型例题和解题方法

### 2. 理解概念本质
- 不要死记硬背公式，要理解定理的证明思路
- 搞清楚各个概念之间的联系和区别
- 尝试用自己的话复述定理内容


### 3. 及时查漏补缺
- 数学知识前后关联性强，前面没学好后面会更吃力
- 遇到不懂的地方及时搞懂，不要堆积
- 定期复习，巩固记忆

## 📝 考试相关

### 常见题型
- 计算题（求最大公约数、模逆元、解同余方程等）
- 证明题（定理证明、性质推导）
- 应用题（结合密码学场景）

### 复习建议
- 重点掌握定理的条件和结论
- 熟练掌握各类计算方法
- 理解证明的思路和逻辑
- 多做历年真题和模拟题
